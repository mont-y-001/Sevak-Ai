Process Flow
Zero friction. Zero app. Zero internet needed.

From a phone call to a matched volunteer in minutes — entirely automated, working on any feature phone.

📋
NGO Registers
One-time web portal registration with Darpan ID. Admin approves within 24 hrs.
📞
AI Calls Weekly
Monday 10 AM — AI calls every coordinator in their language, PIN verified.
🧠
Speech → Data
Gemini transcribes + extracts structured JSON. 70%+ confidence required per field.
✅
Validated & Posted
3-layer validation: range check, anomaly detection, confidence threshold.
🚀
Volunteers Dispatched
AI matches by skill + pincode + availability. WhatsApp notification sent instantly.
Technology
Built entirely on Google Cloud

Every component is a Google technology — making this a true Google Solution Challenge submission.

🤖
Gemini 1.5 Flash
Voice script generation, multilingual conversation, JSON extraction with confidence scoring, volunteer ranking
🎙️
Cloud Speech-to-Text
Multilingual transcription — Hindi, Bengali, Tamil, Telugu, English and 6+ regional languages
🔥
Firebase Suite
Firestore for data, Realtime DB for live dashboard, Auth for OTP login, FCM for push notifications
⚡
Cloud Functions
Serverless validation pipeline, AI matching engine, call orchestration — zero infrastructure management
🗺️
Google Maps Platform
Pincode-level need map on coordinator dashboard, proximity-based volunteer matching
📱
Flutter + Exotel
Cross-platform coordinator & volunteer app (Android + Web). Exotel for Indian telephony outbound calls
